# DECISIONS — Telegram Azan Bot

## Принятые решения
- Источник идентифицируется по username.
- "Имрӯз" отправляется 24/7 до успеха.
- Интервал ретраев: 5 минут.
- Новый день начинается в 02:00 (TZ `+05:00`).
- Валидность ответа определяется датой из текста; время получения сообщения не учитывается.
- Лейблы молитв фиксированы (Субҳ, Қиём, Аср, Шом, Хуфтан).
- Cache минимальный: date + times.
- MQTT republish только при изменении данных.
- Лог‑ротация по размеру < 512KB, путь `/logs`.
- Креды/секреты храним в `.env`, не в спецификациях.
- Зависимости фиксируются через `requirements.txt`.
- `TELEGRAM_CHAT_ID` задается явно в `.env`.
