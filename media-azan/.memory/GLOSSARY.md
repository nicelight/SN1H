# GLOSSARY — Telegram Azan Bot

- Azan: время молитвы.
- MQTT retain: флаг сохранения последнего сообщения в топике.
- QoS: уровень доставки MQTT.
- Source bot: `@vaktiNamozBot` (идентификация по username).
- Cache: локальный файл `today-azan-time.json` с последними данными.
- Secret/env: чувствительные значения через env (например, пароль MQTT).
- TELEGRAM_CHAT_ID: chat_id группы для отправки "Имрӯз".
