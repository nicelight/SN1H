В Kodi перейдите в «Настройки → Службы → Управление» и включите управление по HTTP (или JSON-RPC). Обязательно задайте пароль

Home Assistant может автоматически найти Kodi, если веб-интерфейс Kodi включён. Если Kodi не найден автоматически, настройте интеграцию вручную:
«Настройки → Устройства и службы → Добавить интеграцию», и выбрав Kodi.

В итоге в HA появится новая сущность типа media_player.kodi. 
```
automation:
  - alias: "Запуск трека в 08:00"
    trigger:
      - platform: time
        at: "08:00:00"
    action:
      - service: media_player.play_media
        target:
          entity_id: media_player.kodi
        data:
          media_content_type: "SONG"  # или "music" в зависимости от конфигурации
          media_content_id: "library://music/track/12345"  # замените на ID или путь трека
```

или же

``` 
          media_content_type: DIRECTORY
          media_content_id: special://profile/playlists/video/feuerwehrmann_sam.xsp
```

Если в Kodi для запуска трека требуется использовать метод API, можно заменить действие на вызов kodi.call_method:

```
     action:
      - service: kodi.call_method
        target:
          entity_id: media_player.kodi
        data:
          method: "Player.Open"
          media: 
            media_content_type: "SONG"
            media_content_id: "library://music/track/12345"
```
